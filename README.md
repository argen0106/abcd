#Магазин одежды#

##Перед запуском:##
1. База создана: выполнен скрипт database.sql в Mysql.
2. В db.py указаны верные host, user, password, database.
3. Установлены библиотеки: PyQt6, pymysql

#Запуск: python main.py#

##Учетные записи:##
- admin / admin
- manager / manager
- client / client
- кнопка "Войти как гость"

##Состав проекта:##
- main.py - исходный код приложения
- database.sql - дамп базы данных
- er_model.pdf - ER-модель базы данных
- admin.py - окно администатора
- manager.py - окно менеджера
- client.py - окно клиента
- ghost.py - окно гостя
